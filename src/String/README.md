### 541 reverse-string-II  指定位置字符逆序
eg: Input: s = "abcdefg", k = 2  Output: "bacdfeg"
```
s = list(s)
for i in range(0, len(s), 2*k):
    s[i:i+k] = reversed(s[i:i+k])
    return "".join(s)
```
### 字符串移位
给定一个字符串str，和一个整数i。i代表str中的位置，将str[0...i]移动右侧，str[i+1...N-1]移到左侧。 <br>
eg: str="ABCDE", i=2  output: "DEABC". 要求时间复杂度为O(N), 额外空间复杂度为O(1)。<br>
```

2、将str[i+1...N-1]部分的字符逆序  CBADE -> CBAED
3、将str整体的字符逆序  CBAED -> DEABC
```

### 合法括号序列判断
给定一个字符串str，判断是不是整体有效的括号字符串。 <br>
举例：<br>
str="()",返回true。str="(()())"，返回true。<br>
str="(())"，返回true。<br>
str="())"，返回false。str="()a()"，返回false。<br>

最优解 Time: O(N)  Space: O(1)  <br>
1、整型变量num，代表"("出现次数与")"出现次数的差值。 <br>
2、遍历的过程中如果遇到"("则num++。<br>
3、遍历的过程中如果遇到")"则num--。<br>
4、遍历的过程中如果num<0，说明右括号多于左括号，则直接返回false。<br>
5、如果一直没有出现情况4，则一直遍历下去。<br>
6、遍历完成后，如果num==0，则返回true，否则返回false。<br>

求出以str中每个字符结尾的情况下，最长无重复字符子串的长度，并在其中找出最大值返回。<br>
```
哈希表map —> 其中统计了每种字符之前出现的位置
整型变量Pre -> 代表以s[i-1]结尾的情况下，最长无重复子串的长度。
---------------_c--------
          s[i-1] s[i]
```
